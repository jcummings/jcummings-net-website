Frequently Asked Questions

Creating automations is designed to be intuitive:

  1. Tap the "+" button in the Automations tab
  2. Choose a trigger (time-based, location, app event, etc.)
  3. Add actions from connected services
  4. Set conditions and parameters
  5. Test the workflow before activating

Start with the pre-built templates for common workflows like "Meeting Preparation" or "End of Day Recap."

Security is our top priority for integrations:

  • OAuth 2.0: Industry-standard secure authentication
  • Limited permissions: Only request necessary access scope
  • Token encryption: All authentication tokens are encrypted
  • Revokable access: Disconnect services anytime in Settings
  • Regular security audits: We continuously monitor for vulnerabilities

Your credentials are never stored on our servers - only secure tokens for API access.
